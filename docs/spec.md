# 项目业务规格

## 1. 文档目标

本文档定义外贸电商网站的业务范围、角色、前后台产品模块、页面级能力与验收标准。后续 API 契约、数据库结构与代码实现都必须以本文档为业务真源。

## 2. 项目概述

项目目标是建设一个面向海外工业品客户的 B2B/B2C 混合型电商网站，前台整体视觉与站点编排参考 StepMotech / vexmotor，覆盖商品展示、购物车、下单、会员中心以及询价流程；后台提供完整的商品管理、订单查看和询价处理能力。

## 3. 业务目标

- 支持工业品商品目录化展示与搜索筛选。
- 支持标准购买流程：商品浏览、购物车、结账、订单查询。
- 支持询价商品流程：商品详情页按商品模式切换为询价入口。
- 支持用户中心：资料、地址、订单、询价记录。
- 支持后台完整商品管理，保证前后台数据统一。
- 支持后续扩展多语言、多币种、第三方登录与营销模块。

## 4. 用户角色

### 4.1 游客

- 浏览首页、分类、搜索、商品详情。
- 添加普通商品到购物车。
- 发起询价。
- 注册账号。

### 4.2 注册用户

- 登录/退出。
- 维护个人资料与地址簿。
- 管理购物车并下单。
- 查看订单历史与订单详情。
- 查看个人询价记录。
- 收藏商品。

### 4.3 后台运营人员

- 管理商品、分类、品牌、图片、附件、规格 SKU、库存、价格、SEO、多语言。
- 切换商品为“可购买”或“询价”模式。
- 查看订单与询价列表。
- 维护首页内容位、推荐位、品牌位等可配置内容。

### 4.4 超级管理员

- 拥有全部后台权限。
- 管理后台账号、角色权限与系统配置。

## 5. 业务模式

### 5.1 标准购买模式

- 商品详情页显示价格、库存、规格、数量选择和加入购物车按钮。
- 商品加入购物车后可进入购物车、结账、生成订单。
- 订单支持未支付、已支付、处理中、已发货、已完成、已取消等状态。

### 5.2 询价模式

- 商品被标记为询价商品时，详情页隐藏标准加购区，显示询价入口。
- 用户可以提交姓名、邮箱、电话、公司、国家、需求说明、商品信息等询价内容。
- 询价支持游客和登录用户两种提交方式。
- 提交成功后生成询价记录，后台可查看与跟进。

### 5.3 双模式切换规则

- 每个商品必须有明确的购买模式：`buy` 或 `inquiry`。
- `buy` 模式允许进入购物车与结账。
- `inquiry` 模式不允许进入购物车，但允许显示附件、参数、相关推荐等商品信息。

## 6. 前台产品模块

### 6.1 首页

首页整体视觉参考 vexmotor：

- 顶部公告、主导航、搜索入口、账户入口、购物车入口。
- 首屏大横幅，承载品牌、主打品类与 CTA。
- 分类入口卡片区。
- `Hot Sale` 商品区。
- `New Release` 商品区。
- `Featured Industries` 行业应用区。
- 品牌背书 / 公司介绍区。
- 用户评价 / 证言区。
- Newsletter 订阅区。
- 底部服务卖点：免邮、退换、支付、安全、支持。

### 6.2 分类列表页

- 分类面包屑。
- 分类头图、简介。
- 子分类入口。
- 商品列表网格。
- 排序：综合、最新、价格、热度。
- 筛选：分类、品牌、价格、属性、库存状态。
- 分页。

### 6.3 搜索结果页

- 关键词搜索。
- 结果总数。
- 与分类页一致的排序、筛选、分页体验。
- 无结果兜底内容。

### 6.4 商品详情页

- 商品名、SKU/型号、品牌。
- 图集与缩略图。
- 价格、促销信息、库存状态。
- 规格/属性选择。
- 数量选择。
- 购买模式切换：普通购买或询价。
- 商品说明、规格参数、附件下载、FAQ/扩展标签、相关推荐。
- 收藏、分享、最近浏览等增强能力。

### 6.5 购物车

- 查看购物车商品。
- 修改数量、删除商品。
- 计算小计、运费、税费、总计。
- 进入结账。

### 6.6 结账

- 支持登录用户和游客结账。
- 填写或选择地址。
- 选择配送方式。
- 选择支付方式。
- 确认订单。
- 生成订单详情页。

### 6.7 会员中心

- 我的资料。
- 地址簿。
- 订单列表。
- 订单详情。
- 收藏列表。
- 询价记录。

### 6.8 认证

- 注册。
- 登录。
- 退出登录。
- 找回密码二期考虑。
- 第三方登录作为扩展位保留，首期不启用。

### 6.9 询价

- 询价表单支持商品上下文自动带入。
- 支持列表页/详情页进入询价。
- 支持后台查看处理状态。
- 支持邮件通知作为二期功能。

## 7. 后台产品模块

### 7.1 商品管理

- 商品列表、搜索、筛选。
- 新增、编辑、删除。
- 上架、下架、批量操作。
- 基本信息：名称、slug、摘要、详情。
- 品牌、分类、特征、标签。
- 规格 SKU / 变体。
- 价格、促销价、库存。
- 图片、附件。
- SEO 字段。
- 多语言字段。
- 购买模式切换。

### 7.2 分类管理

- 分类树。
- 分类名称、描述、图片、排序、SEO。
- 启用/禁用。

### 7.3 品牌管理

- 品牌列表。
- 品牌名称、logo、描述、站点展示开关。

### 7.4 订单管理

- 订单列表。
- 订单状态流转查看。
- 收货信息、商品明细、支付信息查看。

### 7.5 询价管理

- 询价列表。
- 按状态筛选。
- 查看客户信息与商品信息。
- 跟进备注与状态更新。

### 7.6 用户管理

- 后台账号与角色二期可扩展。
- 前台会员列表与详情首期只读查看。

## 9. 页面级验收标准

### 9.1 首页验收

- 页面结构完整呈现参考站首页主要模块。
- 首屏、商品区、行业区、订阅区和底部卖点均可配置内容。
- 移动端和桌面端可正常浏览。

### 9.2 商品详情页验收

- `buy` 模式时可加入购物车。
- `inquiry` 模式时不显示标准加购按钮，显示询价入口。
- 图集、价格、规格、附件、相关推荐展示正常。

### 9.3 购物与下单验收

- 普通商品可从详情页加入购物车并完成下单。
- 订单创建后会员中心可查看。

### 9.4 询价验收

- 询价商品可成功提交询价记录。
- 后台可以查询到询价数据。

### 9.5 后台商品管理验收

- 可新增、编辑、删除商品。
- 可维护图片、库存、价格、规格、SEO、多语言、购买模式。
- 前台展示结果与后台配置一致。
